Controls Keyboard

Move
	Walk Up			→	W  /  Axis -1
	Walk Down		→	S  /  Axis +1
	Walk Left		→	A  /  Axis -2
	Walk Right		→	D  /  Axis +2
	Dash			→	Spacebar  /  O

Spells
	Cast Spell		→	Right Click  /  X
	Change Left Spell 	→	Q  /  L1
	Change Right Spell	→	E  /  R1

Others
	Pause			→	Escape  /  Start
	Attack			→	Left Click  /  Square
	Interact		→	F  /  Triangle

Debug Keys
	Show Hitbox		→	F1
	God Mode		→	F10
	Change Player (In HUB)	→	N
